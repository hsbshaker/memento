# Memento Landing Page – Build Spec

## Goal
Build a premium, minimal, high-conviction landing page that clearly communicates value within 5 seconds and drives users to start onboarding.

This is NOT a generic SaaS page. It should feel like a high-end fintech product.

---

## Section Order

1. Top Navigation
2. Hero Section
3. How It Works (3-step section)
4. Final CTA Section

---

## Global Design Principles

- Dark mode only
- Premium, calm, minimal
- Avoid clutter and unnecessary UI
- Strong typography hierarchy
- Subtle gradients and glow (no harsh borders)
- Airy spacing (never cramped)
- No generic dashboard or analytics feel

## Non-Negotiables

- Do NOT change any copy
- Do NOT add extra sections
- Do NOT introduce charts or analytics visuals
- Do NOT make it look like a budgeting app
- Keep everything minimal and premium
